        De nombreuses applications avec lesquelles vous interagissez stockent des données telles qu'un nom d'utilisateur, un mot de passe ou une liste d'amis

        Ce type de données vit dans une base de données . La plupart des bases de données sont des collections de tables stockant différents éléments de données .



        Selon vous, quel type de données une application comme Mimo stocke ?


        SQL est un langage premettant de modifier, d'insérer et d'accéder aux données des tables d'une base de données.

        Qu'est-ce que SQL ?


        Les tableaux sont comme des feuilles de calcul stockant des informations sur un type spécifique de données, comme ce tableau stockant les identifiants, noms et e-mails des utilisateurs.

        userID name email
        1 Sam sam17@mail.com
        2 Remy rem@mail.com
        3 Luis luis_99@mail.com

        Chaque ligne d'un tableau représente un nouvel élément, comme la façon dont la première ligne contient des données sur l'ID, le nom et l'adresse e-mail d'un élément utilisateur.

        userID name email
        1 Sam sam17@mail.com
        2 Remy rem@mail.com
        3 Luis luis_99@mail.com

        Si chaque ligne stocke un élément, combien d'éléments utilisateur cette table stocke-t-elle ?

        userID name email
        1 Sam sam17@mail.com
        2 Remy rem@mail.com
        3 Luis luis_99@mail.com
